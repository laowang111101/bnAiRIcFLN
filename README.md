# 人事管理系统简介 python575

人事管理系统是基于Python语言和Django框架开发的一款Web项目，旨在为公司提供一个高效、便捷的人员管理解决方案。

## 技术栈

- 后端语言：Python
- 项目框架：Django
- 数据库：MySQL 5.7/8.0
- 编译器：PyCharm
- Python版本：3.6.8+
- 前端框架：Vue

## 功能模块

### 系统角色

- 管理员
  - 系统首页
  - 用户管理
  - 员工管理
  - 公司信息管理
  - 招聘信息管理
  - 应聘申请管理
  - 反馈结果管理

- 用户
  - 系统首页
  - 公司信息
  - 招聘信息
  - 培训信息
  - 公告信息

- 员工
  - 系统首页
  - 公司信息管理
  - 招聘信息管理
  - 应聘申请管理
  - 反馈结果管理
  - 培训信息管理
  - 参加培训管理
  - 培训记录管理
  - 考勤信息管理
  - 请假信息管理
  - 加班信息管理
  - 弹窗提醒管理
  - 我的信息

## 运行环境

- Python 3.6.8 或以上版本
- Django 框架
- Vue 前端框架
- MySQL 5.7 或 8.0 数据库
- PyCharm 编译器（推荐）

## 部署步骤

1. 环境准备：安装Python、MySQL、PyCharm等必需软件。
2. 数据库配置：根据需求创建数据库，导入数据库文件。
3. 项目配置：配置项目中的数据库连接、静态资源等。
4. 运行项目：通过PyCharm或者命令行运行项目。

## 目录结构

```
project/
│
├── apps/             # 应用目录
│   ├── admin/        # 管理员相关
│   ├── employee/     # 员工相关
│   └── user/         # 用户相关
│
├── static/           # 静态资源文件

│

```
